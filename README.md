# Dynamic Product Listing Interface

A premium, fully responsive React application that fetches and displays products from the Fake Store API. Built with React.js, Tailwind CSS, and Vite.

## 🚀 Features

- **Premium UI/UX**: Modern design with glassmorphism, smooth animations, and hover effects.
- **Dynamic Data fetching**: Integrated with Fake Store API for real-world product data.
- **Search & Filter**: Real-time product search by title and filter by categories.
- **Sorting**: Sort products by price (Low to High / High to Low).
- **Responsive Layout**: Optimized for Mobile, Tablet, and Desktop.
- **Graceful Error Handling**: Custom loader and error states with retry functionality.

## 🛠️ Tech Stack

- **React 19** (Functional components, Hooks)
- **Tailwind CSS v3** (Custom styling, animations)
- **Vite** (Build tool)
- **Lucide/SVG Icons** (UI elements)

## 📁 Project Structure

```text
src/
├── components/     # Reusable UI components (Navbar, ProductCard, etc.)
├── pages/          # Page-level components
├── services/       # API integration logic
├── utils/          # Helper functions (formatters)
├── App.jsx         # Main application logic
└── index.css       # Global styles and Tailwind directives
```

## 🏁 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

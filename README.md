
# Dynamic Product Listing Interface

A fully responsive, production-ready React application designed to fetch and display product data dynamically using the Fake Store API. This project emphasizes modern UI/UX principles, performance, and maintainable architecture, making it suitable for scalable front-end applications.

## Overview

The application provides users with an intuitive interface to browse, search, and filter products in real time. It demonstrates best practices in component-based architecture, API integration, and responsive design using contemporary front-end technologies.

## Key Features

* **Modern User Interface**
  Clean and visually appealing design incorporating glassmorphism, smooth transitions, and interactive hover effects to enhance user experience.

* **Dynamic Data Integration**
  Seamless integration with the Fake Store API to fetch and render real-world product data dynamically.

* **Search and Filtering**
  Real-time search functionality allowing users to filter products by title, along with category-based filtering for improved navigation.

* **Sorting Capabilities**
  Users can sort products by price in ascending or descending order to quickly find relevant items.

* **Responsive Design**
  Fully optimized layout ensuring consistent performance and usability across mobile, tablet, and desktop devices.

* **Error Handling and Loading States**
  Includes custom loading indicators and robust error handling with retry mechanisms to ensure reliability and smooth user interaction.

## Technology Stack

* **React 19**
  Utilizes functional components and React Hooks for efficient state and lifecycle management.

* **Tailwind CSS v3**
  Provides utility-first styling with support for custom themes, animations, and responsive design.

* **Vite**
  A fast and lightweight build tool for modern web development, enabling rapid development and optimized production builds.

* **Lucide Icons / SVG Assets**
  Used for scalable and consistent UI iconography.

## Project Structure

```text
src/
├── components/     Reusable UI components (e.g., Navbar, ProductCard)
├── pages/          Page-level components and layouts
├── services/       API integration and data-fetching logic
├── utils/          Utility functions and helpers
├── App.jsx         Root component and application logic
└── index.css       Global styles and Tailwind configuration
```

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

### 3. Build for Production

```bash
npm run build
```
